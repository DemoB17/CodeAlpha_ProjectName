# Project Overview

## Introduction

This project was completed as part of the CodeAlpha Data Analytics Internship.

The goal of the project is to collect book information from the Books to Scrape website using Python web scraping techniques. The collected data was cleaned, organized into a structured dataset, analyzed using Microsoft Excel, and visualized to identify trends and patterns.

## Objectives

- Build a web scraper using Python.
- Extract book titles, prices, and ratings.
- Store the data in CSV format.
- Perform exploratory data analysis (EDA).
- Create visualizations to communicate insights.
- Demonstrate practical data analytics and web scraping skills.
